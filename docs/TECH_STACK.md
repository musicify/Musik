# Tech-Stack Entscheidung

## Frontend & Backend
- **Next.js 14+** (App Router)
  - React Framework mit Server Components
  - API Routes für Backend
  - SSR/SSG für Performance
  - TypeScript Support

## Datenbank
- **PostgreSQL**
  - Relationale Datenbank für komplexe Beziehungen
  - Skalierbar und zuverlässig
- **Prisma ORM**
  - Type-safe Database Client
  - Migrations
  - Schema Management

## Authentifizierung
- **NextAuth.js v5**
  - Session Management
  - OAuth Support (Google, etc.)
  - Email/Password Auth

## Styling
- **Tailwind CSS**
  - Utility-first CSS
  - Responsive Design
  - Dark Mode Support
- **shadcn/ui**
  - Komponenten-Bibliothek
  - Accessible Components

## Datei-Upload & Storage
- **AWS S3** oder **Cloudinary**
  - Audio-Dateien
  - Profilbilder
  - Portfolio-Dateien

## Zahlungen
- **Stripe**
  - Kreditkarten
  - PayPal Integration möglich
  - Escrow-System

## E-Mail
- **Resend** oder **SendGrid**
  - Transaktions-E-Mails
  - Benachrichtigungen

## Real-time Chat
- **Socket.io** oder **Pusher**
  - Live-Chat zwischen Kunde und Regisseur
  - Status-Updates

## Audio-Vorschau
- **WaveSurfer.js** oder **Howler.js**
  - Audio-Player
  - Waveform-Visualisierung

## Development Tools
- **ESLint** - Code Quality
- **Prettier** - Code Formatting
- **Husky** - Git Hooks
- **TypeScript** - Type Safety
